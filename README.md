# Proyecto IAS (Front + Back)

Este proyecto está organizado en dos partes principales:

## Estructura de Carpetas

- **frontend/**: Aplicación React con Vite y TailwindCSS.
- **backend/**: Servidor Node.js con Express, configurado para integrar funcionalidades de IA.

## Instalación y Ejecución

### Backend
1. Entra a la carpeta backend: `cd backend`
2. Instala dependencias: `npm install`
3. Crea un archivo `.env` (puedes copiar el ejemplo si lo hay)
4. Ejecuta el servidor en desarrollo: `npm run dev`

### Frontend
1. Entra a la carpeta frontend: `cd frontend`
2. Instala dependencias: `npm install`
3. Ejecuta el servidor de desarrollo: `npm run dev`

## Tecnologías

- **Frontend**: React, Vite, TailwindCSS, React Router
- **Backend**: Express, Cors, Dotenv, (Estructura lista para OpenAI/LangChain)
